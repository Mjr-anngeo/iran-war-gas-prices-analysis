# iran-war-gas-prices-analysis
Exploratory analysis of state-wise gas price changes using Python, with visualizations of regional trends and price increases.
# Iran War Gas Prices Analysis by State

## Project Overview
This project analyzes the impact of Iran war tensions on gas prices across U.S. states using Python.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Jupyter Notebook

## Dataset Features
- State
- Region
- Gas Price on March 19, 2026
- Pre-war Gas Price
- January 08, 2026 Gas Price
- Price Increase Since War
- - Percentage Increase Since War
- Price vs National Average

## Objectives
- Analyze gas price increase by state
- Compare pre-war and post-war prices
- Identify regions with highest gas prices
- Visualize fuel market impact using charts

## Key Insights
- Gas prices increased across many states
- Some states saw sharper rises than others
- Regional differences were clearly visible

## Conclusion
This project shows how geopolitical tensions can affect domestic fuel prices across regions and states.
